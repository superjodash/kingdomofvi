# Kingdom of VI — Lessons (26 Locations)

**Format per location**
- Location ID, Chapter
- Map label
- Unlocks (new keys)
- Prerequisite
- Scroll (modal) copy + vignette ASCII art
- Lesson Arena Buffer (large content)
- Quest (1–3 gem criteria)
- Award copy

> Defaults:
> - **Arrow keys are disabled in lesson arenas** unless explicitly stated.
> - The player can replay completed lessons to improve gem ratings.
